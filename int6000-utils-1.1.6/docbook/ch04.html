Introduction

An Intellon chipset consists of an integral CPU, ROM and proprietary circuitry. The CPU requires a minimum amount of external SDRAM to execute runtime software and store runtime configurtation parameters. The INT6000™ chipset also requires a minimum amount of external NVRAM in order to start. The INT6300™ can use external NVRAM in the same way as the INT6000™ or it can use a local host processor as surrogate NVRAM.

On startup, the SDRAM memory controller must be configured before runtime firmware and parameters are loaded. On the INT6000™, runtime firmware and configuration parameters must be loaded from external NVRAM. On the INT6300™, it may be loaded from external NVRAM or from a external host processor. Runtime firmware determines device capability. Runtime configuration parameters determine device network identity and personality.

The following sections identify and describe firmware related components and discuss some of the routine actions required to manage them. Consult the Intellon HomePlug AV Hardware Technical Reference Manual and HomePlug AV Firmware Technical Reference Manual for more information.
