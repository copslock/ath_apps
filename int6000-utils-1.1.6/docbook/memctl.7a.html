NAME
       memctl - Memory Configuration Applet

SYNOPSIS
       int6ktest memctl.nvm [applet] [...]

PURPOSE
       This configuration applet initializes the SDRAM controller on INT6300 and INT6400 devices.
       It returns to the bootloader on completion. This applet should be downloaded and  executed
       once before downloading and executing applets that test or execute in SDRAM.

       The  advantage  of  using  this  applet is platform independence. On the INT6300, it is an
       alternative to downloading SDRAM Parameters with a VS_SET_SDRAM message and, consequently,
       elminates  the  need for an SDRAM configuration file. On the INT6400, it is an alternative
       to downloading and executing the Denali Applet.

PREREUISITES
       Bootloader must be running.

EXAMPLES
       The  following example uses program int6ktest to download and execute applet file mctl.nvm
       then applet file mytest.nvm.

          # int6ktest memctl.nvm mytest.nvm
          eth1 00:B0:52:00:00:01 Write Memory (00000040:6076)
          eth1 00:B0:52:00:00:01 Wrote memctl.nvm
          eth1 00:B0:52:00:00:01 Start memctl.nvm (0)
          eth1 00:B0:52:00:00:01 Write Memory (00001000:4708)
          eth1 00:B0:52:00:00:01 Wrote mytest.nvm
          eth1 00:B0:52:00:00:01 Start mytest.nvm (0)

       The next example uses program nvmmerge to combine the memctl and nvramtest applet into one
       image file then uses program int6ktest to execute both applets in order at once.

          # nvmmerge memctl.nvm mytest.nvm > favorite.nvm
          # int6ktest favorite.nvm
          eth1 00:B0:52:00:00:01 Write Memory (00000040:6076)
          eth1 00:B0:52:00:00:01 Wrote favorite.nvm
          eth1 00:B0:52:00:00:01 Start favorite.nvm (0)
          eth1 00:B0:52:00:00:01 Write Memory (00001000:4708)
          eth1 00:B0:52:00:00:01 Wrote favorite.nvm
          eth1 00:B0:52:00:00:01 Start favorite.nvm (1)
