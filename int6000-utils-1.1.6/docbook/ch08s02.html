ihpapi_ResetDevice

int ihpapi_ResetDevice (uint8_t  sa [],
 uint8_t  da [],
 size_t  bufferLen,
 uint8_t  buffer []);

Encode a frame buffer as an Intellon vendor-specific managment message that will reset the Intellon powerline device. Return the number of bytes actually encoded on success or 0 on failure. On return, global variable errno will identify the cause of any failure with a POSIX error code. The function is declared in ihpapi.h and defined in ResetDevice.c. See an example of this function in ResetDevice2.c.

Arguments sa and da point to the 6 octet source and destination Ethernet addresses to be used. Argument buffer points to the frame buffer and argument length is the buffer length.
