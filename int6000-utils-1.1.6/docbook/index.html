Abstract

Intellon Corporation, Orlando FL USA, offers the Intellon Linux Toolkit free of charge to their customers. The toolkit contains C Language source code and GNU Makefiles needed to compile dozens of useful programs for engineers and software developers. It also includes a variety of Bash shell scripts that use these programs, in various ways, to interrogate, control and manage Intellon INT6000™, INT6300™ and INT6400™ devices. These programs are simple and compact, making them suitable for embedded applications where resources are at a premium. Most all have command line interfaces, making them generally unsuitable for end users or computer novices.

Although this Toolkit is intended for Linux environments, it includes a Visual Studio 2003 solution file and pre-compiled versions of most programs that are ready to install and run on most Microsoft Windows platforms. The Linux and Windows applications have the same code base but Intellon does not offer technical support for this toolkit on Microsoft Windows. WinPcap™ must be installed in order to use Toolkit programs on Microsoft Windows platforms.

New users should read the entire first chapter of this document before starting to use the toolkit. This will probably answer most questions concerning installation, structure, content and application.
