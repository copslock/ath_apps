AV-ator™ Widget Installation on Windows

Intellon provides an example Yahoo! widget for demonstration purposes. The Intellon AV-ator™widget runs on Windows™ and displays either the transmit or receive PHY rate of a given powerline adaptor using Intellon Toolkit programs. It needs the Yahoo! Widgets SDK™, available from Yahoo!, this is a development version that can be skinned with a custom meter face.

Widget Installation on Windows

Widget installation on Windows is essentially the same as Toolkit Installation on Windows with the addition of the Yahoo! Widgets™ Deskbar and the Intellon AV-ator™ widget, itself.

The AV-ator™ widget is distributed as a WinZip™ archive, named AV-ator-xxx-xx-xx.zip. The file contains all widget source code and images, a Yahoo! Widgets 4.5™ installation program, a DotNet 1.1™ installation program and a WinPcap 4.0.2™ installation program.

  1. Extract the AV-ator archive file into a temporary folder on your Windows host. It will create a folder named AV-ator.widget.

  2. Using Windows Explorer, double-click on file AV-ator.widget/Contents/Installation/AV-ator-install.bat to start installation. The batch file will invoke Control Panel applets and installation files in proper sequence. If problems occur during installation, merely restart the installation at this step and skip the steps that have been successfully completed. Most of the steps only need to be done once.

  3. The DotNet 1.1 ™ installation file, dotnetfx.exe, will execute. If you do not have this specified version installed then install it, even if you have a later version. Old and new version can coexist. Otherwise, close the installation dialogue and continue.

  4. The WinPcap 4.5™ installation file, WinPcap_4_0_2.exe, will execute. It you do not have this version, or a later version, installed then install it. Otherwise, close the installation dialogue and continue.

  5. The Control Panel Add/Remove Software applet will open to show installed applications. If you have a previous version of the Intellon Toolkit installed then remove it. Otherwise, close the dialogue and continue.

  6. The Intellon Toolkit installation file, int6000-utils-x.x.x.msi, will execute. If you do not have this version of the toolkit installed then install it. Otherwise, close the dialogue and continue.

  7. The Control Panel System applet will open. Select the Advanced tab then press the Environment Variables pushbutton.

  8. If folder C:\Program Files\Intellon\Toolkit is not in your current Path string then add it. If you have Administrator permissions then edit the System Variable list. Otherwise, edit the User Variable list. If you do not know how to do this then ask someone. Close the dialogue to continue.

  9. Internet Explorer will open. If you do not have Yahoo! Widgets 4.5™ installed then download and install it. Otherwise, close the download dialogue and continue.

  10. The Yahoo! Widgets Deskbar will appear from the right or left side of you screen as the cursor approaches the edge. Click on the gear icon in the upper left of the Deskbar and select Open Widget ... from the menu that appears. Select file AV-ator.kon to load and execute the AV-ator™ widget. If asked to copy the widget, respond in the affirmative.

  11. The first time the widget executes, you will be prompted with a confirmation dialogue. After that, the widget will appear as a pushbutton on the Deskbar.

  12. Once the AV-ator™ appears, right-click on the meter face to open the preferences menu. Select an Ethernet interface from the dropdown Host Interface list and powerline device from the Powerline Link list. Press the Save pushbutton to save your preferences. If there are any powerline devices connected to the selected interfaces then the meter needle will indicate the PHY rate.

Widget Removal on Windows

After successful installation, you may delete the original archive file and the temporary folders that it created. The installed AV-ator™ widget will be located in your My Documents\My Widgets\AV-ator.widget folder. It can be removed from there using the Yahoo! Widgets™ Deskbar.
