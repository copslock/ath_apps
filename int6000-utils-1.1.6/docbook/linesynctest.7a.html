NAME
       linesynctest - Powerline Synchronization Diagnostic Applet

SYNOPSIS
       int6ktest linesynctest.nvm [...]

PURPOSE
       This diagnostic applet runs in SDRAM. It checks onboard zero-cross detection circuitry and
       reports the powerline frequecy. Possible outcomes are 50hz detected, 60hz detected, unsup‚Ä‚Äê
       ported frequency detected and no frequency detected.

       Progress  and  status  messages  appear  on the local host console. Control returns to the
       bootloader on completion.

PREREQUISITES
       Bootloader software must be running. The SDRAM controller need not be configured.

EXAMPLES
       The following example uses program int6ktest to download and execute applet  linesync.nvm.

          # int6ktest linesynctest.nvm
          eth2 00:B0:52:00:00:01 Start Applet linesynctest.nvm (0)
          eth2 00:B0:52:00:00:01 Write Memory (00001000:1800)
          eth2 00:B0:52:00:00:01 Wrote linesynctest.nvm
          eth2 00:B0:52:00:00:01 (00) Line Sync test - Zero Cross detected - 60HZ
