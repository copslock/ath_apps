Internal API Functions

EncodeEthernetHeader

size_t EncodeEthernetHeader (uint8_t  buffer [],
 signed  length,
 uint8_t  ODA [],
 uint8_t  OSA []);

Encode a buffer of specified length in bytes with a standard Ethernet header. The encoded header will contain the specified destination address ODA and source address OSA and the HomePlug® AV Ethertype constant HOMEPLUG_MTYPE. Return the number of bytes actually encoded or 0 on encoding error. Set global variable errno to an appropriate POSIX error code defined in errno.h. The function is declared in ihp.h and defined in EncodeEthernetHeader.c. Constant HOMEPLUG_MTYPE is defined in ihp.h.

EncodeIntellonHeader

size_t EncodeIntellonHeader (uint8_t  buffer [],
 signed  length,
 uint16_t  MMTYPE);

Encode a buffer of specified length in bytes with an Intellon vendor-specific message header. The encoded header will contain the HomePlug® AV protocol version constant HOMEPLUG_MMV and the specified Intellon message type MMTYPE. Return the number of bytes actually encoded or 0 on encoding error. Set global variable errno to an appropriate POSIX error code defined in errno.h. The function is declared in ihp.h and defined in EncodeIntellonHeader.c. Constant HOMEPLUG_MMV is defined in ihp.h.
