NAME
       mrgpib - Intellon PIB Merge Utility

SYNOPSIS
       mrgpib [options] target source

DESCRIPTION
       (DO NOT USE THIS PROGRAM TO MERGE PIBS 4-0 OR LOWER WITH PIBS 4-1 OR HIGHER.)

       Copy  information  from  one  PIB  file to another while preseving PIB version, length and
       checksum. The program automatically adjusts for differences in PIB version and  length  so
       don’t worry. Be happy!

       This  program  is part of the Intellon Linux Toolkit. See the ilt man page for an overview
       and installation instructions.

OPTIONS
       -q     Suppress progress messages. Does nothing for this application.

       -v     Print additional information. Does nothing for this application.

       -?,--help
              Print program help summary on stdout.  This  option  takes  precedence  over  other
              options on the command line.

       -!,--version
              Print  program  version  information  on  stdout. This option takes precedence over
              other options on the command line. Usethis option  when  sending  screen  dumps  to
              Intellon  Technical  Support  so  that they know exactly which version of the Linux
              Toolkit you are using.

ARGUMENTS
       target The target PIB file. Information in this file will be replaced  with  corresponding
              information  in  the  source file. If the source file is longer that this file then
              excess information is not copied; otherwise, all source file information is copied.
              No assumptions are made based on filename and no filename conventions are enforced.

       source The source PIB file. Information in this file will replace  corresponding  informa‐
              tion  in the target file. If the target file is longer that this file, all informa‐
              tion will be copied; otherwise, excess information is not  copied.  No  assumptions
              are made based on filename and no filename conventions are enforced.

EXAMPLES
       The following example copies information from file old.pib to file new.pib.

          # mrgpib new.pib old.pib
