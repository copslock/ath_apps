NAME
       connection - Intellon Connection Analyser

SYNOPSIS
       int6ktest connection.nvm [...]

PURPOSE
       This  diagnostic  applet  runs  in Tightly Coupled Memory (TCM) and performs a sequence of
       memory read/write tests that detect bad circuit board connections. Control returns to  the
       bootloader on successful completion.

FAILURE
       All  LEDs  will  flash  on  failure. If this test fails the most other tests will probably
       fail.  Look for broken connections, cold solder joints and shorted pins.

PREREQUISITES
       The device must be in bootloader mode.
